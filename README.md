## How to Play

1. **Tap a gem** to select it
2. **Tap an adjacent gem** to swap them
3. **Match 3 or more** gems of the same type horizontally or vertically
4. **Score points** and watch the gems cascade down
5. You have **30 moves** to get the highest score possible!
